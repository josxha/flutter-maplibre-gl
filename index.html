Getting Started

Customizable Maps

maplibre_gl supports the highly customizable Mapbox Vector Tiles (mvt) standard. You can customize the map style, add markers, and much more.

High Performance

Reliable and fast, maplibre_gl is built on top of the MapLibre GL JS for Web and MapLibre Native for iOS and Android to achieve native speed.

Open-Source and Vendor-Free

MapLibre provides a collection of open-source projects that have no vendor lock-in. You can freely choose your map data provider or host your own tiles.
